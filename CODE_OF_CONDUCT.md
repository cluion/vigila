# 行為準則

## 我們的承諾

為營造開放友善的環境，我們承諾讓每位參與者的貢獻與交流不因年齡、體型、身心障礙、族群、性別認同與表達、經驗程度、國籍、外貌、種族、宗教或性取向而受到騷擾。

## 行為準則

有助於營造正向環境的行為包括：

- 使用友善包容的語言
- 尊重不同觀點與經驗
- 樂於接受建設性批評
- 以社群整體利益為重

不被接受的行為包括：

- 帶有性意味的言語或影像、不受歡迎的關注
- 挑釁、侮辱或貶抑性言論，人身或政治攻擊
- 公開或私下騷擾
- 未經許可公布他人私人資訊

## 執行

違反行為準則者，專案維護者有權移除、編輯或拒絕不符本準則的留言、commit、程式碼、issue 與其他貢獻。

如遇濫用、騷擾或其他不當行為，可透過 GitHub 的私密回報功能聯繫維護者。所有申訴都會被審視與調查。
